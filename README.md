# Password Generator 🔐

A Python-based password generator that creates strong, unique passwords.

## Features
- Alphanumeric + special characters
- Never repeats a password
- Minimum 13 characters

## Requirements
- Python 3.x

## How to Run
python password_generator.py

## Usage
1. Enter desired password length (min 13)
2. Enter how many passwords to generate
3. Copy and use your secure passwords!
